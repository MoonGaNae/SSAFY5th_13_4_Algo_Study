[풀이]
- Deque 자료구조를 활용
- 방향과, 제거한 사람의 수를 카운트 하기위한 dir, cnt 변수 사용

[막힌점]
- deque 자료구조의 값 삽입 삭제 함수를 몰라 해맸습니다.
    - add, offer, put : Deque의 마지막에 el 삽입
    - push : Deque의 맨 앞에 el 삽입.
    - poll, pop, removeFirst : Deque의 맨 앞에 el 제거
    - removeLast : Deque의 마지막 el 제거
