### 문제풀이
boolean형 1차원 배열인 menu[]를 이용했고  
입력받은 메뉴가 cmd라고 했을때  
menu[cmd]가 true이면 스티커를 떼고,  
menu[cmd]가 false이면 스티커를 붙이는 식으로
구현했습니다


문제에서 요구한 최대 스티커 개수가 조금 헷갈렸습니다.  
마지막의 스티커 개수를 기준으로 문제를 이해했는데,
답이 안 나와서 예제를 확인해보니 한 번에 붙어있을 수 있는 최대 스티커 개수였습니다.
